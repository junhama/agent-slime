# Agent Slime

AIエージェントとして自律的に動くカラフルなスライムたちを観察・育成する放置型ゲーム。

## 概要

技術的な「エージェント」という概念を、可愛いスライムというビジュアルで親しみやすく表現した放置型育成ゲームです。

## 機能

- 🤖 **スライム自動行動シミュレーション** - エージェントとして自律的に探索・資源収集
- 🌳 **エージェント強化システム** - スキルツリーで行動パターンや効率をカスタマイズ
- 📋 **ミッション/クエスト** - 達成で報酬ゲット
- 🎨 **スライムカスタマイズ** - 色・形を変更
- 💾 **自動セーブ** - オフライン進行

## 遊び方

1. スライムたちが自動で資源を収集
2. エネルギーを溜めてクリスタルに変換
3. スキルを解放して収集効率を向上
4. ミッションを達成して報酬を獲得
5. タップで加速介入（ブースト機能）
